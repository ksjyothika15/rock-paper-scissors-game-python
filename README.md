# 🎮 Rock Paper Scissors Game

A simple command-line **Rock Paper Scissors** game built using **Python**. The game allows a user to play against the computer, where the computer makes random choices using Python's `random` module.

---

## 📖 About the Project

This is my **first Python mini project** uploaded to GitHub. It demonstrates the use of Python fundamentals such as loops, conditional statements, lists, user input, and the random module.

---

## ✨ Features

- 🎲 Computer generates a random choice
- 👤 User vs Computer gameplay
- ✅ Determines the winner correctly
- ⚠️ Handles invalid input
- 🔁 Supports multiple rounds

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/ksjyothika15/rock-paper-scissors-game-python.git
```

2. Open the project folder.

3. Run the program:

```bash
python rock_paper_scissors.py
```
